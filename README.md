# IoT-Based Quality Control System Using Camera Sensors and Deep Learning for Defect Detection

This repository contains the source code and resources for the research paper:
**"IoT-Based Quality Control System Using Camera Sensors and Deep Learning for Defect Detection"**

---

## üöÄ Overview
This project proposes an **IoT-enabled deep learning system** for real-time defect detection in manufacturing lines.  
It integrates **camera-based sensors**, **Convolutional Neural Networks (CNNs)**, and **IoT cloud dashboards** for intelligent quality control.

---

## üß© Features
- Real-time defect detection using CNN
- IoT integration with ThingSpeak/MQTT for remote monitoring
- Cloud-based visualization dashboard
- Low latency (<250ms) and high accuracy (~96.84%)

---

## üß† Tech Stack
- **Python 3.10**
- **TensorFlow / Keras**
- **OpenCV**
- **Flask (for dashboard API)**
- **ThingSpeak Cloud / AWS IoT**
- **ESP32-CAM + Raspberry Pi 4**
